Exercicíos de fixação de Banco de Dados
1. o modelo relacional de banco de dados foi desenvolvido em 1970
2. Edgar Frank Codd é considerado o "pai dos bancos de dados relacionais"
3. banco de dados é uma "coleção de dados" armazenados em um repositório contendo fatos relacionados a algo, o SGBD é o programa responsável por gerenciar esses bancos de dados. O sistema de banco de dados integra ambos.
4. características do banco de dados: padronização de do acesso, segurança do acesso, integridade das informações, escalabilidade, trabalho em equipe.
5. os níveis de abstração em um banco de dados são: modelo conceitual, modelo lógico e modelo físico.
6. as vantagens de um banco de dados são:
- organização dos dados
- segurança
- evita a redundância de dados
